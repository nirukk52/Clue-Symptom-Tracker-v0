# Brand Guidelines

Visual and voice guidelines for Chronic Life marketing.

## Visual Identity

### Colors

| Name          | Hex       | CSS Variable      | Usage                        |
| ------------- | --------- | ----------------- | ---------------------------- |
| Primary       | `#20132E` | `--primary`       | Text, buttons, dark elements |
| Background    | `#FDFBF9` | `--bg-cream`      | Main background              |
| Accent Peach  | `#E8974F` | `--accent-peach`  | Selection states, CTAs       |
| Accent Blue   | `#A4C8D8` | `--accent-blue`   | Decorative blobs             |
| Accent Purple | `#D0BDF4` | `--accent-purple` | Highlights, glow effects     |
| Accent Mint   | `#B8E3D6` | `--accent-mint`   | Success states               |
| Accent Rose   | `#F4C4C4` | `--accent-rose`   | Gentle warnings              |
| Text Muted    | `#554b66` | `--text-muted`    | Secondary text               |

### Typography

| Use     | Font     | Weight  | Example                      |
| ------- | -------- | ------- | ---------------------------- |
| Display | Fraunces | 400-700 | Headlines, emotional moments |
| Body    | DM Sans  | 400-600 | All other text               |

### Spacing & Radius

- Section gaps: `space-y-8`
- Element gaps: `space-y-4`
- Padding: `p-6` (containers), `px-4 py-3` (buttons)
- Border radius: `rounded-2xl` (cards), `rounded-pill` (buttons)

## Voice & Tone

### We Sound Like

- A friend who gets it
- Calm and reassuring
- Direct without being clinical
- Empathetic but not pitying

### We Don't Sound Like

- A doctor or medical professional
- Corporate or sterile
- Overly cheerful or dismissive
- Every other health app

### Voice Examples

**Good:** "Log the flare without explaining your whole life"
**Bad:** "Comprehensive symptom logging for chronic conditions"

**Good:** "20 seconds. That's it."
**Bad:** "Quick and easy symptom tracking"

**Good:** "No vibes. Only receipts."
**Bad:** "Evidence-based insights"

## Visual Style

### Do

- Warm, calming backgrounds
- Soft gradients and gentle shadows
- Fintech-like premium clarity
- Plenty of white space
- Subtle animations (float, pulse)

### Don't

- Medical imagery (hospitals, pills, stethoscopes)
- Stock photo aesthetic
- Harsh contrasts
- Cluttered layouts
- Illness-focused imagery

## Logo Usage

- Logo: "C" mark + "Chronic Life" wordmark
- Always use dark navy on light backgrounds
- Minimum clear space: 1x logo height

## Material Icons

Use Material Symbols Outlined:

- `favorite` - Brand/heart moments
- `bolt` - Quick actions
- `smart_toy` - Assistant
- `local_fire_department` - Flare mode
- `history` - History view
- `psychology` - Insights
